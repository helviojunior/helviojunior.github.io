Segue um passo a passo para instalação do PHP5.6 no Ubuntu

Preparando Ambiente

Primeiro passo é remover a versão atual do PHP

[sourcecode language="shell"]sudo apt-get remove --purge php5*
[/sourcecode]

Depois, adicionar o repositório do PHP5.6 e atualizar a listagem de pacotes

[sourcecode language="shell"]sudo add-apt-repository ppa:ondrej/php
sudo apt-get update
[/sourcecode]

Instalando PHP5.6 + NGINX

Adicione o repositório do NGINX e atualize a listagem de pacotes

[sourcecode language="shell"]echo deb http://nginx.org/packages/mainline/ubuntu/ `lsb_release --codename --short` nginx >> /etc/apt/sources.list
sudo apt-get update
[/sourcecode]

E por fim instale o NGINX + PHP-FPM

[sourcecode language="shell"]sudo apt-get install nginx php5.6-common php5.6-cli php5.6-fpm
[/sourcecode]

Instalando PHP5.6 + Apache2

Recomendo a utilização do ambiente com NGINX, por ter menos falhas e um desempenho muito melhor. Mas caso precise/deseje utilizar o Apache basta instalar com os comandos abaixo:

[sourcecode language="shell"]apt-get install apache2 php5.6-common php5.6-cli php5.6 libapache2-mod-php5.6
[/sourcecode]

Biblioteca para MySQL

Caso vá utilizar o MySQL como base de dados basta adicionar o pacote do MySQL

[sourcecode language="shell"]sudo apt-get install php5.6-mysql
[/sourcecode]

Pronto!
