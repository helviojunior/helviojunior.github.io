A SafeTrend, lançou um novo plugin para integração entre o SafeId e o Zabbix.

Com essa integração é possível fazer de forma simples alguns processos de administração de usuário, como por exemplo, é possível importar usuários de um arquivo texto ou excel e integra-los com o Zabbix, além de poder definir perfis de acesso.

Segue abaixo a lista de algumas das possibilidades proporcionadas por essa integração:

- Criação, modificação e remoção de usuário
- Single Sign-On com outros sistemas
- Criação de grupos
- Atribuição de usuário ao grupos
- Definição de permissões
- Replicação/troca de senha
- Controle de acesso por horário (definir qual horário o usuário pode acessar o zabbix)

Caso tenha alguma dúvida segue o site do SafeId

[http://single-sign-on.com.br/](http://single-sign-on.com.br/)

The SafeTrend, build a new plugin to integrate SafeId and Zabbix.

This integration help-us in zabbix user management, for example, with this is possible to import into zabbix an users list from excel, also create users grups and profiles.

Folow some possibles with this integration:

- Create, change and delete user
- Single Sign-On
- Create groups
- Add user in group
- Set user permissions
- Replication of password
