Este post foge um pouco do que comumente posto, que é relacionado a TI e fotografia, mas como tive bastante dificuldade para achar este esquema estou postando aqui.

O objetivo deste post é mostrar o esquema elétrico do som (imagem abaixo) utilizado em alguns carros da Citroen e Peugeot. A Motivação deste post foi a necessidade de instalar um Subwoofer no Citroen C4.

[![Som]({{ site.baseurl }}/assets/2015/10/Som.png)]({{ site.baseurl }}/assets/2015/12/Som.png)

<!--more-->

Alguns termos que me ajudaram no processo de busca na internet.

Geralmente este aparelho tem uma etiqueta na lateral conforme a imagem abaixo:

[![Etiqueta_som]({{ site.baseurl }}/assets/2015/10/Etiqueta_som-300x169.jpg)]({{ site.baseurl }}/assets/2015/12/Etiqueta_som.jpg)

Este som (RD4) utiliza o padrão de conector, conforme a imagem abaixo:

[![008_RD4_Todas_Vias_s]({{ site.baseurl }}/assets/2015/10/008_RD4_Todas_Vias_s-300x225.jpg)]({{ site.baseurl }}/assets/2015/12/008_RD4_Todas_Vias_s.jpg)

Este conector utiliza a seguinte pinagem:

[![anschluss]({{ site.baseurl }}/assets/2015/10/anschluss.jpg)]({{ site.baseurl }}/assets/2015/12/anschluss.jpg)

Onde:

| **Pino** | **Função** |
| --- | --- |
| **Part A** |  |
| 1 | + Audio Traseiro Direito |
| 2 | + Audio Dianteiro Direito |
| 3 | + Audio Dianteiro Esquerdo |
| 4 | + Audio Traseiro Esquerdo |
| 5 | - Audio Traseiro Direito |
| 6 | - Audio Dianteiro Direito |
| 7 | - Audio Dianteiro Esquerdo |
| 8 | - Audio Traseiro Esquerdo |
| 9 | Não utilizado |
| 10 | Dados CAN (high) *** |
| 11 | +12v remoto, controle para ativar amplificadores/módulos/antenas |
| 12 | +12v alimentação do radio |
| 13 | Dados CAN (low) *** |
| 14 | Controle para mute |
| 15 | Não utilizado |
| 16 | GND (mass), ground |
| **Part B** |  |
| 1 | + Audio Direito AUX 2 |
| 2 | - Audio (GND) AUX 2 |
| 3 | Não utilizado |
| 4 | + Audio Direito AUX 1 |
| 5 | Não utilizado |
| 6 | Não utilizado |
| 7 | + Audio Esquerdo AUX 2 |
| 9 | Microphone |
| 10 | + Audio Esquerdo AUX 1 |
| 11 | - Audio (GND) AUX 1 |
| 12 | Não utilizado |
| **Part C** |  |
| 1 | GND, ground para CD-Charger |
| 2 | Sinal CAN para CD-Charger |
| 3 | + bateria para CD-Charger |
| 4 | + Audio Direito do CD-Charger |
| 5 | Não utilizado |
| 6 | Não utilizado |
| 7 | Dados CAN (high) *** |
| 8 | - Audio (GND) do CD-Charger |
| 9 | Não utilizado |
| 10 | + Audio Esquerdo do CD-Charger |
| 11 | Não utilizado |
| 12 | Não utilizado |
